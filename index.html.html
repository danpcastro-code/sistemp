Manual de Equipe

Configurao do Backend

1

Preparar Banco

No Supabase, use o 'SQL Editor' para rodar o script de criao da tabela 'sistemp_data'.

2

Coletar APIs

Em 'Project Settings' > 'API', copie a URL do projeto e a chave anon public.

3

Hospedar Site

Clique no boto preto 'Baixar Sistema' e coloque o arquivo gerado em um servidor (Vercel/Netlify).

4

Distribuir

Envie o link do site para sua equipe. Eles devem colar as chaves apenas uma vez.
